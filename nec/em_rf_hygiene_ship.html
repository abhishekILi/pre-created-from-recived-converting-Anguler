2. Top Deck
Ser Check Item Yes/No Remarks
1Are all movable / hinged structures (hatches and doors) bonded with Type I bond straps?
2Are all Type I bond straps free from rust and corrosion?
3Are all removable / collapsible structures bonded using Type II bonds?
4Are all Type II bond straps free from rust and corrosion?
5Have all metallic boxes on the top deck been bonded to the deck?
6Are all unscreened cables on top deck routed through metallic conduits / trunks?
7Are cable screens / conduits bonded circumferentially (360 degrees) at penetration points?

11Are there any loose metallic items in vicinity of radar and communication antennas?
