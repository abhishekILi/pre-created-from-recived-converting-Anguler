Ser Reference Checks Remarks
5.1 Chapter-III Para 1 of EED-50-48 Check Switchboard layout in APMS. It should be as per approved GRAQ/ATP/BPs.
5.2 Chapter-III Para 26 of EED-50-48 Monitor the PGD configuration: The APMS shall monitor and clearly indicate the following configuration information:
5.2.1 Close Ring / Open Ring
5.2.2 Fwd / Aft split
5.2.3 Port / Stbd split
5.2.4 Island Mode
5.3 Chapter-III Para 17 and Chapter-IV Para 6 of EED-50-48 Check MIMIC Pages on APMS Console. The system should represent the entire network on Multi-Function Consoles (MFCs) using hierarchical mimic diagrams.
5.4 Chapter-III Para 4 of EED-50-48 Check Power Management Modes of APMS System. Auto Mode, Assisted Mode, Manual Mode and Training Mode.
5.5 Chapter-III Para 11 of EED-50-48 Check Maximum Load Setting for Generators. The software should permit change in maximum load limit for the individual generator.
5.6 Chapter-III Para 12 of EED-50-48 Check Central Control and Switching. The system should permit the following:
5.6.1 Switching ON/OFF any of the breakers of the main switchboards from the control consoles (Fwd/Aft Switchboard).
5.6.2 Switching ON/OFF of main EDC breakers (if exists) from control consoles that would be located in main switchboards.
5.6.3 Remote switching ON/OFF of generators from the console provided in the MCR.
5.6.4 Monitoring the status of individual MCCBs of the EDCs (if exists) from the control consoles.
(List of breakers as per class of ship, all breakers to be tested for remote operation)
5.7 Chapter-III Para 14 and Chapter IV Para 12 of EED-50-48 Check Failure Alarms and Indications on APMS Console. The system should provide extensive audio/visual failure alarms and indications to ensure safe operation.
5.8 Chapter-III Para 13 of EED-50-48 Check Feasibility of Parallel Operation with Shore Supply. Provision should exist to monitor the parameters of the shore supply so that changeover takes place smoothly and safely.
5.9 Chapter-III Para 15 and Chapter IV Para 13, & 14 of EED-50-48 Check Data Logging & Data Archiving Facility Available in APMS. The system should have the facility of data logging and archiving facility for APMS and Interface systems for a minimum of 30 days.
5.10 Chapter-III Para 23 of EED-50-48 Check Post Failure Review Facility Available on APMS Console. The system should be able to undertake Post-Disturbance-Review to detect exact reasons of failure in an interconnected system.
5.11 Chapter-III Para 22 of EED-50-48 Check Onboard Training System (OBTS) on APMS Console. The System should have a mode of operation that will enable training for operator and maintainer on the system. In this mode of operation, the ship's machinery will not be controlled by the functions that are executed during the training.

5.12.{{ forloop.counter }} {{ label }}
5.13 Chapter-III Para 3-29 of EED-50-48 Check Auto Mode of APMS System. The system would carry out all functions automatically without human intervention. Following checks to be performed in Auto Mode:
5.13.1 Take control of APMS in Auto mode.
5.13.2 Check generator priority selection as per requirement/availability.
5.13.3 Check Auto load dependent Start/Stop of DGs.
5.13.3.1 If Load > 80% (for 2.5 secs) on running DA – Priority 1 generator should automatically start and taken on load. In case, priority 1 generator fails to start, priority 2 generator should start and taken on load. In case, priority 2 generator fails to start, then priority 3 generator should start and taken on load. If priority 3 generator fails to start, then next priority generator (if available) should start and taken on load.
5.13.3.2 If Load < 25% (for 15 minutes) on running DA, priority-based Generator should be off loaded after transferring load to the running generators. The generator will however be required to be stopped manually by the operator.
5.13.4 Check synchronization of Generators. The system should automatically parallel the generators and effect the load transfer.
5.13.5 Check Active/Reactive Load Sharing during parallel operation. System should ensure proportionate active/reactive load sharing by the running generators when in continuous parallel mode operation.
5.13.6 Check Black Out Handling operation of APMS: On detecting the loss of an alternator or complete loss of shore supply, APMS shall take following actions:
5.13.6.1 Ensure that all the switchboards are in Island operation.
5.13.6.2 Start next priority Alternator and connect the respective busbar.
5.13.6.3 Once Alternator start operation is complete, evaluate the system configuration and re-establish the configuration in effect before black out.
5.13.6.4 If the configuration cannot be established then provide a suitable prompt to the operator.
Note: This check will be undertaken for all generators by selecting priority 1 in sequential manner and other generator may be used as running generator.
5.13.7 Check Preferential/Automatic load shedding: (In case failure of one running generator leading to overloading another running generator or the load exceeds on the standalone generator)
5.13.7.1 Adequate audio/visual warning should be given prior to load shedding.
5.13.7.2 If the running generator is overloaded and no standby generator is nominated, the system should resort to automatic load shedding as per pre-defined criteria/priority selection of breakers.
5.13.8 Check Control transfer from FWD Console to AFT Console and vice versa.
5.14 Chapter-III Para 3-29 of EED-50-48 Check Auto Assisted (Semi Auto) Mode of APMS system. All actions are required to be carried out by the operator as prompted by the system. Following checks to be performed in Auto Assisted Mode:
5.14.1 Take control of APMS in Auto-Assisted mode.
5.14.2 Select generator priority selection as per requirement/availability.
5.14.3 Check Remote load dependent Start/Stop of DGs.
5.14.3.1 If Load > 80% (for 2.5 secs) on running DA – the system should initiate audio and visual alarms so that the operator can take necessary action to start standby generator.
5.14.3.2 If Load ≤ 25% (for 15 minutes) on running DA – The system should give necessary audio/visual alarms to alert the operator to off load and shut down the under loaded generator. However, if the generators are running in parallel, the system should ensure their equal loading.
5.14.4 Check synchronization of Generators. The system should automatically synchronize the generators and prompt the operator for closure of the supply breaker.
5.15.1 Check Active/Reactive Load Sharing during parallel operation. System should ensure proportionate active/reactive load sharing by the running generators when in continuous parallel mode operation.
5.15.2 Check Black Out Handling operation of APMS. On detecting the loss of an alternator or complete loss of shore supply, APMS shall take following actions:
5.15.2.1 Ensure that all the switchboards are in Island operation.
5.15.2.2 Shows popup window on APMS console to Start next priority Generator and connect the respective busbar by operator.
5.15.2.3 Once Alternator start operation is complete, evaluate configuration and re-establish previous configuration.
5.15.2.4 If configuration cannot be established then provide a suitable prompt to the operator.
Note: This check will be undertaken for all generators by selecting priority 1 in sequential manner and other generator may be used as running generator.
5.15.2.5 Check preferential load shedding (in case failure of one running generator leading to overloading another running generator or the load exceeds on the standalone generator).
5.15.2.6 Adequate audio/visual warning should be given prior to load shedding.
5.15.2.7 System should recommend loads to be shed as per pre-defined criteria/priority selection of breakers.
5.15.3 Check Control transfer from FWD Console to AFT Console and vice versa.
5.16 Chapter-III Para 31 of EED-50-48 Check UPS for APMS System.
5.16.1 Must be provided for all MFCs, Control units and interfacing
5.16.2 Backup — Minimum 30 minutes
